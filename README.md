# transport_action_cam_data
- アクションカムのデータを外付けHDDに移すデスクトップアプリケーション
  - 撮影日を取得して自動的にコピー先に撮影日毎にディレクトリを作って振り分ける

# 主要機能
## 動画・画像コピー機能
- SDカード等に保存されている動画/画像ファイルを、撮影日時に基づいて年月日フォルダ構造で整理
- 任意のコピー元フォルダから任意の保存先フォルダへ整理可能
- `YYYY/MM/DD/` フォルダ構造で自動作成・配置

## 動画・画像プレビュー機能
- 選択されたコピー元ディレクトリや選択されたコピー先ディレクトリの内部にある動画や画像を閲覧可能
