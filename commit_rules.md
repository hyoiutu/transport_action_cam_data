# Commit Rules (コミット規約)

本ファイルは、AIエージェントが自動コミットスキルを実行する際に遵守すべき詳細なルールを定義します。

---

## 🚫 絶対遵守ルール (Mandatory Rules)

1. **プッシュの絶対禁止**
   - 本スキルまたはAIエージェントの処理を通じて、リモートリポジトリへのプッシュ (`git push`) を絶対に実行してはならない。

2. **テスト・リントの無視禁止**
   - コミット前に実行したテストやリントが失敗した場合、その結果を無視してコミットを進めてはならない。
   - 失敗した場合はエラー内容をユーザーに報告し、修正を行うか、コミット処理を中断すること。

3. **コミットメッセージの要約**
   - コミットメッセージは変更内容を要約し、1行程度（簡潔で明確）にまとめること。

4. **コミット前のユーザー確認**
   - コミットを実行する前に、以下の情報を必ずチャットに明示し、ユーザーから Y/N の承認を得ること。
     - コミットされるファイル一覧
     - コミットメッセージ案
     - 対象ブランチ名
